# 💰 Smart Expense Splitter

A mobile application that helps groups of friends **track and split shared expenses easily**.
Users can add expenses, divide costs among participants, and calculate who owes whom.

This project is built using **React Native with Expo**, enabling cross-platform support for Android and iOS.

---

## 🚀 Features

* Add and manage group expenses
* Split bills among multiple people
* Automatic balance calculation
* Track who owes whom
* Clean and simple mobile UI
* Cross-platform support (Android / iOS)

---

## 🛠 Tech Stack

* **React Native**
* **Expo**
* **JavaScript / TypeScript**
* **Expo Router**
* **Node.js / npm**

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/Smart_Expense_Splitter.git
```

Navigate to the project folder:

```
cd Smart_Expense_Splitter
```

Install dependencies:

```
npm install
```

---

## ▶️ Run the Application

Start the Expo development server:

```
npx expo start
```

You can run the app on:

* Android Emulator
* iOS Simulator
* Expo Go mobile app
